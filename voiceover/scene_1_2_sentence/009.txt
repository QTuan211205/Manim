There are three main ways to use test-time compute. First, we can make the model generate extra tokens, such as in chain-of-thought: the model writes intermediate reasoning steps before giving the final answer. Second, we can call the generator multiple times. Third, we can move from a single language model to a compound AI system. In such a system, the model can be combined with evaluators, verifiers, code interpreters, search engines, or other external tools.
